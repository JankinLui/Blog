    # 2.2.3 inotifywait命令使用方法

    -m| --monitor                      ---实现一直监控目录的数据变化
-r |  --recursive                  ---进行递归监控
-q | --quiet                       ---尽量减少信息的输出
--format <fmt>                     ---指定输出信息的格式
--timefmt <fmt>                    ---指定输出的时间信息格式
-e | ---event                      ---指定监控的事件信息
