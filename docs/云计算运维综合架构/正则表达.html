    # 第一章 正则符号

    1. 利用正则符号匹配文件内容信息
    2. 正则符号经常出现在各种开发语言中，python Java php
    3. 正则符号经常被三剑客命令使用

    基础正则符号：basic regular expression RBE

    # 1.1 尖号符号：^

    以什么开头的信息 ll /oldboy | grep "^d"

    # 1.2 美元符号：$

    以什么结尾的信息 ll -F /etc/ | grep /$

    # 1.3 尖号美元：^$

    空行

    # 1.4 点：.

    匹配任意一个且仅有一个字符

    # 1.5 星符号：*

    匹配前一个字符连续出现0次或者多次

    # 1.6 点星符号：.*

    匹配任意信息 类似于通配符*

    # 1.7 反斜杠：\

    将有意义的字符信息，转义成普通字符进行识别

    --> 找出文件以点结尾的信息

    将特殊的扩展正则符号，转化成基础正则符号

    将没有意义的字符信息转化成有意义的字符

    # 1.8 中括号：[ ]

    匹配多个字符信息，多个字符匹配时，是或者的关系

    [OK] 在文件中找有O的字符信息 找文件中有K的字符信息

    ps： 将一个字符串信息，进行拆分成一个一个的字符进行匹配

    # 1.9 尖号中括号：[^]

    [^ok] 在文件中找有o的字符信息 找文件中有k的字符信息，进行排除不要显示，显示其他信息

    扩展正则符号：

    # 1.10 加号符号：+

    匹配一个或多个先前字符

    将输出的信息echo 123456 ==> 转换为<123456>进行显示

    # 1.11 竖线符号：|

    并且符号，用于匹配多个信息

    14点的failed信息

    grep "14:00|failed" ljq.log
    1

    # 1.12 小括号：（）

    将多个字符信息汇总成整体，进行过滤

    进行后向引用前项

    # 1.13 括号符号：{}

    可以指定字符连续匹配的次数 x{n，m} x{n} x{n，} x{，m}

    表示前一个字符至少连续出现n次，最多出现m次

    # 1.14 问好符号：？

    定义匹配前一个字符出现0次或1次

    总结：

    * 匹配前一个字符出现0次或者多次

    + 匹配前一个字符出现1次或者多次

    {} 匹配前一个字符指定出现多少次，至少次
