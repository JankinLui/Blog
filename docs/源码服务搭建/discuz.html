    Discuz搭建

    贪吃偷懒的猫 2021/9/18

    # 简介

    CrossdayDiscuz! Board（简称 Discuz!）是北京康盛新创科技有限责任公司推出的一套通用的社区论坛软件系统。自2001年6月面世以来，Discuz!已拥有15年以上的应用历史和200多万网站用户案例，是全球成熟度最高、覆盖率最大的论坛软件系统之一。目前最新版本Discuz! X3.3正式版于2017年1月1日发布，是 X3.2 的稳定版本。

    # 第一步：安装环境

    ​ 安装Apache、MySQL、PHP ​ yum install httpd php php-fpm php-mysql mysql mysql-server -y ​ （centos7版本中的MySQL安装方式与6版本不同：yum install mariadb*） ​

    启动服务
6版本:
service httpd start
systemctl start httpd.service
service mysqld start 

7版本:
systemctl start mariadb.service
service php-fpm start             
systemctl start php-fpm.service

    # 第二部：配置数据库

    ​ 配置MySQL ​ mysqladmin -u root password "XXXXXXXX" (此处的密码可进行自定义）设置好 MySQL 的密码后， 对账号密码进行验证 ​ mysql -u root -p输入刚刚设定好的密码，可以登录到 MySQL 中，则说明配置正确。退出 MySQL

    检查LAMP环境配置

    vim /etc/www/html/test.php
    1

    ​ 中添加

    <?php echo "<title>Test Page</title>"; phpinfo() ?>
    1

    输入完成后，按“Esc”键，输入 “:wq”，保存文件并返回 http:// IP/test.php 出现PHP页面,则说明 LAMP 环境配置成功

    # 第三步：安装Discuz

    1.下载安装包

    wget http://download.comsenz.com/DiscuzX/3.2/Discuz_X3.2_SC_UTF8.zip
    1

    2.解压安装包

    unzip Discuz_X3.2_SC_UTF8.zip
    1

    3.把解压后的 “upload” 文件夹下的所有文件复制到 “/var/www/html/”。

    cp -r upload/* /var/www/html/
    1

    4.将写权限赋予给其他用户。这些目录文件上传到服务器之后，默认只有 root 用户才有写权限。

    chmod -R 777 /var/www/html
    1

    5.至此，论坛已经完全搭建完毕，可以在浏览器中进行安装了。
