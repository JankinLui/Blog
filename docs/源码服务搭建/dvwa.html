    DVWA搭建

    贪吃偷懒的猫 2021/9/18 源码服务搭建LinuxDVWA

    # 第一章 DVWA安装

    下载dvwa压缩包

    Wget https://github.com/ethicalhack3r/DVWA/archive/master.zip
    1

    # 1.1 安装所需环境

    sudo yum install -y httpd mysql-server mysql php php-mysql ph
    1

    注：php-gd 库：gd 库是 php 处理图形的扩展库，gd 库提供了一系列用来处理图片的 API，使用 GD 库可以处理图片，或者生成图片。 在网站上 GD 库通常用来生成缩略图或者用来对图片加水印或者对网站数据生成报表及验证码

    全部设置开机启动，然后启动服务

    # 1.2 测试 LAMP 环境:

    vim /var/www/html/test.php
    1
    <?php         phpinfo(); 
?>
    1
    2

    访问 http://192.168.10.9/test.php 正常，说明 LAMP 环境支持 PHP 解析没有问题

    # 1.3 配置 mysql 数据 root 用户密码

    mysqladmin -u root password "123456"
    1

    注：若已经设置 mysql 数据 root 用户密码，则更改：

    mysql -u root –p687145
    1
    mysql> set password=password(‘123456’); 
mysql> quit
    1
    2
    mysql -u root -p123456
    1
    mysql> show databases;
    1

    将下载的 dvwa 渗透系统代码上传到 Linux 上，并解压到网站根目录下

    unzip -d /var/www/html/  DVWA-1.9.zip 
ls /var/www/html/ 
#DVWA-1.9  test.php 
chown apache:apache /var/www/html/DVWA-1.9/ -R
    1
    2
    3
    4

    编辑/dvwa/config/config.inc.php,配置数据库信息，user和password是MySQL的用户名和密码。

    vim /var/www/html/DVWA-1.9/config/config.inc.php  #修改的地方如下:
    1
    1 $_DVWA[ 'db_server' ]   = '127.0.0.1'; 
2 $_DVWA[ 'db_database' ] = 'dvwa'; 
3 $_DVWA[ 'db_user' ]     = 'root'; 
$_DVWA[ 'db_password' ] = '123456';     #只需要修改成你的 mysql 的 root 用户密码
    1
    2
    3
    4

    # 1.4 部署 DVWA 网站系统

    在浏览器中输入：http://192.168.112.63/DVWA-1.9/setup.php

    # 第二章 解决报错信息

    改：

    815行 allow_url_include = Off  为： allow_url_include = On

    service httpd restart
