    ■(2)配置基于（使用）IP地址的虚拟主机

    虚拟主机，是指可以在Apache服务器中架设多个网站。

    ①配置基于IP不同、端口号相同的虚拟主机

    Step1 创建两个虚拟网卡、设置好不同的IP；最好ping一下，确保设置的IP都能用。

    ifconfig eth0:1 192.168.112.129 [netmask 255.255.255.0]
ifconfig eth0:2 192.168.112.130 [netmask 255.255.255.0]
ping 192.168.112.129
ping 192.168.112.130

    3
    4

    Step2 修改Apache服务器的配置文件/etc/httpd/conf/httpd.conf，在里面添加多个虚拟主机的设置

    1003 #<VirtualHost *:80>

1004 #  ServerAdmin webmaster@dummy-host.example.com

1005 #  DocumentRoot /www/docs/dummy-host.example.com

1006 #  ServerName dummy-host.example.com

1007 #  ErrorLog logs/dummy-host.example.com-error_log

1008 #  CustomLog logs/dummy-host.example.com-access_log common

1009 #</VirtualHost>

1011 <VirtualHost 192.168.112.129>

1014 <VirtualHost 192.168.112.130>

1015   DocumentRoot /var/www/html/web2

1016 </VirtualHost>

Step3 分别在/var/www/html目录下创建两个虚拟网站所设置的根目录web1、web2，并且创建相应的首页文件
