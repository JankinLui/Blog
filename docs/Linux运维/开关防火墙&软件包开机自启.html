    开关防火墙&软件包开机自启

    贪吃偷懒的猫 2021/9/17 Linux， Linux防火墙Linux软件包

    注意：

    setenforce 0 # 设置SELinux 成为permissive模式 （关闭SELinux） setenforce 1 # 设置SELinux 成为enforcing模式 （开启SELinux）

    修改配置

    vi /etc/selinux/config
#SELINUX=enforcing
#注释掉
#SELINUXTYPE=targeted
#注释掉
SELINUX=disabled
#增加
:wq! #保存退出
setenforce 0     # 关闭SELinux(临时)

#systemctl disable firewalld  --永久关闭防火墙
#setenforce 0               --临时关闭SELiunx

#vim /etc/seliunx/config      --永久关闭seliunx
SELIUNX=disabled         //将enforcing改为disabled

    5. 关闭SELinux
